# Быстрый старт

## Важно: Для совместного редактирования используйте ASGI-сервер!

Django `runserver` не поддерживает WebSocket. Для работы совместного редактирования нужно запускать сервер через `daphne`.

## 1. Установка зависимостей

### Backend
```bash
pip install -r requirements.txt
```

### Frontend
```bash
cd frontend
npm install
```

## 2. Настройка базы данных

```bash
python manage.py migrate
```

## 3. Запуск

### Терминал 1 - Backend (Django с поддержкой WebSocket)
```bash
# Для совместного редактирования используйте daphne:
daphne -b 0.0.0.0 -p 8000 sheets_project.asgi:application

# Или используйте готовый скрипт:
./start_server.sh
```

**Важно:** Django `runserver` не поддерживает WebSocket! Для совместного редактирования обязательно используйте `daphne`.

Сервер будет на http://localhost:8000

### Терминал 2 - Frontend (React)
```bash
cd frontend
npm start
```
Приложение будет на http://localhost:3000

## 4. Использование

1. Откройте http://localhost:3000
2. Зарегистрируйтесь или войдите в систему
3. Создайте новую таблицу
4. Кликните на ячейку для редактирования
5. Введите значение или формулу (начинается с `=`)

### Совместное редактирование

Для совместного редактирования:
1. Убедитесь, что сервер запущен через `daphne` (см. раздел 3)
2. Откройте одну и ту же таблицу в разных браузерах/вкладках (под разными пользователями)
3. Изменения будут синхронизироваться в реальном времени
4. Вы увидите активных пользователей в верхней части редактора

### Примеры формул:
- `=A1+B1` - сложение
- `=SUM(A1:A5)` - сумма диапазона
- `=AVERAGE(A1:A5)` - среднее значение
- `=MAX(A1:A5)` - максимальное значение
- `=MIN(A1:A5)` - минимальное значение
- `=COUNT(A1:A5)` - количество чисел
